Aggiunge un parametro di versione ai file CSS e JS per evitare problemi di caching.

== Descrizione ==

Il plugin "CSS Versioning" è uno strumento essenziale per i gestori di siti WordPress che desiderano assicurarsi che le modifiche ai loro fogli di stile CSS e script JS siano riconosciute e applicate immediatamente dai browser dei visitatori. Questo plugin facilita l'aggiornamento automatico dei file CSS e JS, evitando problemi di caching e assicurando che i visitatori del tuo sito vedano sempre la versione più recente dei tuoi stili e script.

**Caratteristiche Principali:**

- Aggiornamento Automatico del Versioning: Con un semplice clic, la versione dei tuoi file CSS e JS viene incrementata automaticamente, assicurando che i cambiamenti siano immediatamente visibili.
- Facilità di Uso: Grazie alla sua interfaccia semplice e intuitiva, il plugin è accessibile anche a utenti senza competenze tecniche avanzate.
- Integrazione con WordPress: Il plugin è progettato per lavorare in armonia con qualsiasi tema o plugin, garantendo che le modifiche al CSS e JS siano sempre riconosciute, indipendentemente dalla configurazione del tuo sito.

== Installazione ==

1. Carica la cartella `css-versioning` nella directory `wp-content/plugins/` del tuo sito WordPress.
2. Attiva il plugin tramite il menu "Plugin" in WordPress.
3. Vai alla pagina di impostazioni del plugin per iniziare ad utilizzarlo.

== Uso ==

Dopo aver attivato il plugin:

1. Vai alla pagina di impostazioni del plugin, accessibile dal menu principale di WordPress.
2. Clicca sul pulsante "Aggiorna" per incrementare automaticamente la versione dei tuoi file CSS e JS.
